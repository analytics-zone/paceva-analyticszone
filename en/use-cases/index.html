Use Cases

Use Paceva when you want to answer practical training questions from files you control.

Post-session review

Check intensity, laps, pacing, heart-rate drift, W' depletion, and data quality before deciding what the session means.

Weekly load balance

See how swim, bike, run, walking, hiking, and recovery contribute to acute load, chronic load, fatigue, and freshness.

Race block control

Track threshold history, sport-specific fatigue, and readiness during a triathlon or endurance build.

Private file audit

Open exported workouts locally to inspect missing sensors, suspicious GPS, or mismatched pool lengths before sharing anything.

Optional coach export

Create a selected package for a coach only when you choose. Summaries should come first; raw samples require explicit selection.
